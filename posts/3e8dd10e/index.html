Win10系统安装指南

发表于2021-04-05|更新于2021-05-02|Onektasの各种笔记
|字数总计:921|阅读时长:3分钟

前言

小学一年级是我第一次正式接触电脑，那时用的应该是Windows2000；小学三年级家里买了电脑，用的是WindowsXP，同年也学会了重装系统，这项实用的技术至今从没让我失望过。装机10周年，写篇笔记记录一些宝贵经验


准备工作

推荐准备一个8G以上的U盘，使用U盘安装Windows，当然也可以直接安装，完美主义者表示因为U盘安装可以分区和格式化硬盘，确实比直接重装更加干净


选择系统

        关于专业版、家庭版这类的选择，如果可选，推荐安装专业版，会多很多平常用不到但万一就是会用到的功能。
        冷知识：家庭中文版和中文家庭版是两个完全不同的版本。前者是多语言版选择中文，后者是单语言版，万一出现这两个选择，最好选择前者

装系统第一步————理清自己的需求，选择合适的系统版本。

  1. 最新版
    官网下载工具，推荐U盘方式安装
  2. 其他版
    MSDN下载所需镜像，使用UltralISO软碟通将镜像刻录至U盘
  3. 长期服务版
    MSDN，目前最新的是基于1809的LTSC，下载完后打开双击复制所有内容至U盘，无需刻录

以上三种方式均可直接安装，推荐使用U盘。U盘文件系统使用FAT32，分配单元大小默认。U盘内容做好备份，一旦刻入系统，原先文件将会丢失


开始安装

  1. 安装过程需注意的是分区。如果希望分区显示整数，请使用“分区整数计算器”，随便搜一个即可

  2. 引导部分按照自己习惯设置

  3. 个性化部分可以把广告相关选项全部关闭


系统设置

        进入系统后可以去各官网下载驱动程序，也可以开机后保持联网，Win10会自动下载匹配的驱动，游戏玩家最多手动安装一下显卡驱动就好了

推荐设置

  1. 关闭休眠模式          # 保护硬盘
  2. 关闭快速启动          # 保护硬盘
  3. 关闭系统恢复          # 保护系统
  4. 关闭蓝牙、摄像头          # 用的时候再开
  5. 关闭各种同步和设备历史记录          # 需要就保持开启
  6. 关闭提高指针精准度          # 游戏玩家应该晓得

  1. Win+V打开剪切板          # 方便实用
  2. 设置Pin码登录          # 方便实用
  3. 设置英语输入法          # 方便实用

  1. DX修复工具增强版          # 可以修复很多软件游戏启动问题
  2. BandZip          # 用过的最好的压缩软件
  3. NET3.5          # 极少数情况下会用到

其他

        我每次在学校机房都会进行的操作

  1. 关闭提高指针精准度
  2. 设置中英文输入法，关闭第三方输入法
  3. 整理桌面图标，将浏览器和要用的软件添加到快速启动栏
  4. Win+V打开剪切板
  5. 若第一次使用这台电脑，调整显示器亮度色彩
